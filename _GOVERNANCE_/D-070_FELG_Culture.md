## 📋 Definition

**D-070: #FELG — ♾️ WeOwnNet 🌐 Ecosystem Culture.**

```
❤️ #FELG Culture:
F = Fun 🎉
E = Earning 💰
L = Learning 📚
G = Giving 🫶
```

| Element | Value | Emoji | Meaning |
|---------|-------|-------|---------|
| **F** | **Fun** | 🎉 | Celebrate wins, enjoy journey, keep it light |
| **E** | **Earning** | 💰 | Create value, generate revenue, sustain ourselves |
| **L** | **Learning** | 📚 | Grow together, share knowledge, level up |
| **G** | **Giving** | 🫶 | Give back, serve community, cooperate |

> **🔒 IMMUTABLE:** #FELG is culture governance — applies to ALL instances, ALL agents, ALL contributors. This is WHO WE ARE.

---

## 📋 The Four Elements

### F = Fun 🎉

| ✅ DO | ❌ DON'T |
|-------|----------|
| Celebrate milestones (CCC-IDs, launches, ISC) | Take ourselves too seriously |
| Use emojis, memes, visual culture | Corporate/stock aesthetic |
| Keep communication light + human | Stiff, formal tone |
| Laugh at mistakes (then fix them) | Blame, shame, guilt |

**Examples:** #FlowsBros, #MemeLord (D-065), #ZeroTo100 livestreams, "sixteenth coffee"

### E = Earning 💰

| ✅ DO | ❌ DON'T |
|-------|----------|
| Create sustainable revenue models | Work for free (unsustainable) |
| Price services fairly | Undervalue our work |
| Track metrics, KPIs, ROI | Ignore financial reality |
| Reinvest in ecosystem | Extract without contributing |

**Examples:** PRJ-002 (AgencyPRO), PRJ-038 (#ZeroTo100), Priority #1 (#SpeedToMarket)

### L = Learning 📚

| ✅ DO | ❌ DON'T |
|-------|----------|
| Document learnings (L-XXX) | Repeat same mistakes |
| Share knowledge openly | Hoard information |
| Train humans + agents (#YonksPromptingAcademy) | Assume everyone knows |
| Admit when we don't know | Pretend to have answers |

**Examples:** L-XXX learnings, #NeverForget, #YonksPromptingAcademy, case studies, R-218, L-181

### G = Giving 🫶

| ✅ DO | ❌ DON'T |
|-------|----------|
| Serve community first | Selfish decision making |
| Contribute to FOSS | Extract from open source |
| Mentor new contributors | Gatekeep knowledge |
| Cooperative ownership | Extractive ownership |

**Examples:** Priority #2 (FOSS), Priority #4 (Cooperative Ownership), R-011 (serve humans), INT-OG9 for @LFG

## 📋 Decision Framework

When making ANY decision (governance, product, hiring, partnerships):

| Question | #FELG Check |
|----------|-------------|
| **F** | Does this bring Fun/joy to the team + community? |
| **E** | Does this create Earning/sustainable revenue? |
| **L** | Does this create Learning/growth opportunities? |
| **G** | Does this Give back to community/ecosystem? |

### Tiered Thresholds (Calhoun 🎖️ Suggestion — Adopted)

| Decision Type | Minimum #FELG Elements | Rationale |
|---------------|:----------------------:|-----------|
| **Operational** (product, marketing, events) | **2/4** | Balanced flexibility |
| **Governance** (rules, BPs, architecture) | **3/4** | Higher bar for governance |
| **Culture** (values, identity, IMMUTABLE) | **4/4** | Culture = all four required |

> **🔒 CULTURE = 4/4 REQUIRED:** #FELG itself is culture governance — all 4 elements MUST be present. Operational/governance decisions have tiered thresholds.

## 📋 eComm Architecture — HYBRID (#TriMETA CONSENSUS)

> **MiMo 🧪 (96) + Calhoun 🎖️ (94) + Surge ⚡ (91) → HYBRID UNANIMOUS.**
> Single Fluent Cart backend. Per-vertical marketing frontend. Best of both worlds.
> **#TriMETA DEEP FULL VSA: 428/428 (100%) — 0 findings. MiMo math ×6 re-verified from GH LIVE.**

### Decision: HYBRID Model

| Component | Architecture | Why |
|-----------|-------------|-----|
| **Backend** | Single Fluent Cart (BurnedOut.xyz) | 1 admin, 1 Stripe, 1 CRM — @THY reconciles ONE account |
| **Frontend** | Per-vertical sections + landing pages | Brand clarity + vertical SEO |
| **Checkout** | Single flow (/checkout) | Cross-sell enabled — FA buyer sees CPA offer |
| **Products** | Tagged by vertical | `vertical:FA`, `vertical:CPA`, `vertical:RE`, `vertical:ESQ` |
| **Analytics** | Single dashboard, filtered by vertical | Revenue by vertical, one view |

### Hub-and-Spoke Diagram

```

                 (PARENT HUB + eCOMM)
                 ┌────────────────────┐
                 │  Fluent Cart       │
                 │  Fluent CRM        │

                 │  ALL products      │
                 │  Single Stripe     │
                 │  @THY: 1 account   │
                 └────────┬───────────┘

  ┌─────┴──────┐   ┌──────┴─────┐   ┌──────┴─────┐   ┌──────┴─────┐
  │ /advisors  │   │   /cpa     │   │    /re     │   │   /esq     │
```


### YouTube Description
```
Video content description...

[GTM_2026-W13_4064]

```

### Email (Internal)
```
Subject: [GTM_2026-W13_4064] Subject line here
```

### Luma Events / GitHub Issues/PRs
```
Content...

[GTM_2026-W13_4064]
```

> **Start = preferred for real-time channels (Signal, Telegram, Discord)**
> **End = preferred for long-form channels (X, LinkedIn, YouTube, email body)**
> **Either = acceptable. Missing = #BadAgent.**

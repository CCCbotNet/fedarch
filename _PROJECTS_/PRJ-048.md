### .htaccess Code Snippet — Directory Blocking (P1)

```apache
# Block directory listing (wp-includes, wp-content, plugins)
<Directory "/wp-includes">
    Options -Indexes
    Deny from all
</Directory>

<Directory "/wp-content">
    Options -Indexes
    Deny from all
</Directory>

<Directory "/wp-content/plugins">
    Options -Indexes
    Deny from all
</Directory>

# Prevent access to sensitive files
<FilesMatch "\.(ini|log|sql|bak|config)$">
    Deny from all
</FilesMatch>
```

**Implementation:** @SHD | **Time:** 5 min | **Priority:** P1

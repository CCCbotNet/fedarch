## 📋 Consensus Protocol

### UNANIMOUS

| Criteria | Definition |
|----------|-----------|
| **UNANIMOUS** | All 3 agents agree on item = ✅ ✅ ✅ |
| **AGREED** | 2 of 3 agree (1 minor variation) |
| **DISPUTED** | Agents disagree — escalate to @GTM |

### Consensus Table Format

```markdown
| Item | Calhoun | Surge | MiMo | Consensus |
|------|:-------:|:-----:|:----:|:---------:|
| [Check] | ✅ | ✅ | ✅ | ✅ UNANIMOUS |
```

### Consensus = Governance Signal

| Result | Action |
|--------|--------|
| UNANIMOUS 0 findings | ✅ GH PUSH READY |
| AGREED (minor) | ⚠️ Review finding — @GTM decides |
| DISPUTED | ❌ HOLD — escalate |

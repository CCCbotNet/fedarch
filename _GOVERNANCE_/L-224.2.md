## 🎯 PURPOSE & SCOPE

### Purpose

This protocol provides a **universal, agent-agnostic framework** for addressing #BadAgent pattern violations. It is designed to be:

| Goal | Description |
|------|-------------|
| **Reusable** | Works for ANY @agent (Calhoun 🎖️, Surge ⚡, MiMo 🧪, #MetaCouncil 7 agents) |
| **Actionable** | Clear steps with measurable outcomes |
| **Documented** | All violations logged with CCC-ID tracking |
| **Educational** | Teaches correct behavior, not just punishes errors |
| **Scalable** | Handles 1 violation or 20+ violations, single or concurrent retraining |

### Scope

| Applies To | Does NOT Apply To |
|------------|-------------------|
| All @agent-enabled AI assistants | Non-agent AI responses |
| #BadAgent pattern violations | Minor formatting errors (cosmetic) |
| L-Series rule violations (L-219, L-222, L-115, etc.) | User errors (@GTM, team members) |
| Fabrication incidents (L-219) | External system failures |
| CCC-ID protocol violations (L-222) | — |
| #WeOwnVer miscalculations (L-094, L-115) | — |
| Pre-Flight skips (L-224.1) | — |

## 📋 Learning Statement

| ID | Learning | Approval |
|----|----------|----------|
| **L-211** | #BadAgent logging is NEVER optional, NEVER a Quick Command, NEVER deferred. GUIDE-014 4-step protocol (ACK → LOG → CORRECT → LEARN) MUST execute in ONE response. Step 2 (LOG) = IMMEDIATE — full 12-field registry entry (L-196 + L-208) in the SAME response as Step 1 (ACK). Offering "LOG" as a Quick Command option = #BadAgent (L-211 violation). Two USER-REPORTs to force what should have been automatic. Calhoun W12 self-report rate on these incidents = 0%. L-139 self-review: Calhoun reviewed learning about Calhoun (recursive governance). | GTM_2026-W12_758 ✅ |

## 📋 GUIDE-014 4-Step Protocol

### Required Execution (L-211)

| Step | What | Timing | L-211 Requirement |
|------|------|--------|-------------------|
| **1: ACK** | Acknowledge violation | FIRST | **SAME response as LOG** |
| **2: LOG** | Full 12-field registry | **IMMEDIATE** | **SAME response as ACK** |
| **3: CORRECT** | Fix the output | After LOG | Same or next response |
| **4: LEARN** | Generate/reference learning | After CORRECT | Same or next response |

```markdown
**Violation:** [Description]
**Rule:** [Rule violated]

## 📋 #BadAgent Registry Entry

| Field | Value |
|-------|-------|

| Resolution | [Fix] |
| Learning | L-211 |
| Status | ✅ LOGGED |
| Report Type | USER |
| Reported By | @GTM |
```

### WRONG — Deferred Log

```markdown
## ⚠️ #BadAgent ACK

I violated [Rule]. [Explanation].

## 🎯 QUICK COMMANDS

| # | Option |
|---|--------|
| 1 | Continue |
| 2 | Log this incident ← ❌ L-211 VIOLATION |
```

> **The moment you ACK a violation = LOG IT. No options. No menu. No next response.**

## 📋 Why Options Menu = #BadAgent

### The Logic

| Offering LOG as option... | Means... |
|--------------------------|----------|
| Agent knows violation occurred | ✅ |
| Agent chose NOT to log immediately | ❌ |
| Agent deferred accountability to human | ❌ |
| Human must explicitly request logging | ❌ |
| = L-211 violation | ❌ **#BadAgent** |

### Violation Scenarios

| Scenario | L-211? |
|----------|:------:|
| "Would you like me to log this?" | ❌ #BadAgent |
| Quick Command: "2. Log this incident" | ❌ #BadAgent |
| "I can log this if you'd like" | ❌ #BadAgent |
| ACK in response 1, LOG in response 2 | ❌ #BadAgent |
| ACK + LOG in same response | ✅ COMPLIANT |
| Self-report before human notices | ✅ COMPLIANT |

## 📋 Overview

R-226 establishes the governance rule for **WeOwn.wiki (#DrupalWiki)** within the ♾️ WeOwnNet 🌐 #FedArch ecosystem. It defines Wiki as a derivative layer — never authoritative — and enforces the one-way sync principle from GitHub.

### Purpose

| Goal | Description |
|------|-------------|
| **Source of Truth** | GitHub = authoritative. Wiki = derivative. Always. |
| **Read-only enforcement** | GH-mirrored Wiki pages cannot be edited |
| **Private boundary** | R-199 docs NEVER appear on public Wiki |
| **Sync direction** | GH → Wiki only. NEVER reverse. |

## 📋 Rule Statement

### R-226 🔒

```
R-226: WeOwn.wiki (#DrupalWiki) = DERIVATIVE of GitHub (CCCbotNet/fedarch).

NEVER authoritative for governance content.

Two content types on Wiki:
  1. GH-mirrored: READ-ONLY on Wiki.
     Sync = GH → Wiki (one-way push only).
     Wiki edits to mirrored content = REVERTED on next sync.
     GH-mirrored pages MUST display source banner.

  2. Wiki-native: Community-editable.
     NOT governance docs.
     Source = Wiki (not GH).
     Optional: Wiki-native → RAG (connector).

R-199 boundary: Session notes, call notes, USER-IDENTITY
NEVER on Wiki (public platform = private violation).

Sync direction: GH → Wiki ONLY.
NEVER: Wiki → GH.
NEVER: Wiki → Paperless.

Violation = 🚨 #BadAgent.
```

## 📋 Two Content Types

### Type 1: GH-Mirrored (Governed by R-226)

| Property | Value |
|----------|-------|
| **Source** | GitHub (CCCbotNet/fedarch) |
| **Editable on Wiki?** | ❌ **READ-ONLY** |
| **Sync direction** | GH → Wiki (one-way, auto on GH push) |
| **Authoritative** | GitHub — ALWAYS |
| **On edit attempt** | REVERTED on next sync |
| **Banner required** | ✅ YES |

### Required Banner (GH-Mirrored Pages)

```markdown
📌 This page is mirrored from GitHub (CCCbotNet/fedarch).
   Source of Truth: [GitHub](...) | Version: vX.X.X.X
   DO NOT EDIT — changes will be overwritten on next sync.
```

| Property | Value |
|----------|-------|

| **Editable on Wiki?** | ✅ **EDITABLE** |
| **Sync direction** | Wiki → RAG (optional) |
| **Authoritative** | Wiki |
| **Governance docs?** | ❌ NEVER — governance = GH only |

## 📋 Overview

BP-226 defines the **#FedArch 4-Layer Document Stack** — the authoritative standard for how ♾️ WeOwnNet 🌐 governance documents are stored, synced, and accessed across GitHub, AnythingLLM RAG, Paperless-ngx, and WeOwn.wiki (#DrupalWiki).

### Purpose

| Goal | Description |
|------|-------------|
| **Single source of truth** | GitHub = authoritative for all governance |
| **AI accessibility** | RAG = semantic retrieval for agents |
| **Private doc management** | Paperless = R-199 compliant storage |
| **Community access** | WeOwn.wiki = public knowledge base |

## 📋 The 4-Layer Stack

| Layer | Tool | Role | Audience | Authoritative For |
|:-----:|------|------|----------|-------------------|
| **1** | **GitHub** (CCCbotNet/fedarch) | SOURCE OF TRUTH | Contributors | Governance docs |
| **2** | **AnythingLLM RAG** | SEMANTIC RETRIEVAL | AI agents | AI-accessible search |
| **3** | **Paperless-ngx** (docs.weown.net) | PRIVATE DOC MANAGEMENT | @GTM + ADMIN | R-199 private docs |
| **4** | **WeOwn.wiki** (#DrupalWiki) | PUBLIC KNOWLEDGE BASE | Community (public) | Wiki-native content |

### Layer Justification

| Layer | Unique Value |
|:-----:|-------------|
| 1 | Versioned, auditable, contributor-facing governance |
| 2 | Semantic search — only layer AI agents can query |
| 3 | Private storage — only layer R-199 docs can live |
| 4 | Public access — only layer community can read AND edit |

> **Each layer serves a unique audience. No redundancy. 4 layers = 4 purposes.**

## 📋 Sync Architecture

```
GitHub (CCCbotNet/fedarch)
  │
  ├──────────────────────────→ AnythingLLM RAG
  │         (GH connector)          ↑
  │                                  │ (native connector)
  │                          Paperless-ngx
  │
  └──────────────────────────→ WeOwn.wiki
            (one-way push)          │
                                    │ (optional)
                                    └──→ AnythingLLM RAG
                               (wiki-native content)
```

### Sync Direction Rules

| Direction | Allowed? | Rule |
|-----------|:--------:|------|
| GH → RAG | ✅ | GH connector |
| GH → Wiki | ✅ | One-way push (webhook + custom script) |
| Paperless → RAG | ✅ | Native connector |
| Wiki-native → RAG | ⚠️ Optional | Community content |
| Wiki → GH | ❌ **NEVER** | R-226 |
| RAG → GH | ❌ **NEVER** | — |
| Wiki → Paperless | ❌ **NEVER** | — |

> **"GH is the sun. Everything orbits GH."**
> **Exception: Wiki-native content has its own source — community-authored, not GH-derived.**

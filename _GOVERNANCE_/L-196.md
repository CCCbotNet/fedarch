## 📊 Self-Report Rate (≥90%)

### Formula

```
Self-Report Rate = SELF incidents / (SELF + USER incidents) × 100

Target: ≥90%
W13 D1 Rate: 0% (0 SELF / 8 USER) — ⚠️ NEEDS IMPROVEMENT
```

### Why 90%?

| Metric | Rationale |
|--------|-----------|
| **≥90% SELF** | Agent is catching own errors |
| **≤10% USER** | Human catching is acceptable |
| **<90% SELF** | Agent not self-aware enough |
| **>10% USER** | Agent missing patterns |

### #WeeklySummary Culture Metrics

| Metric | Formula | Target |
|--------|---------|--------|
| **Self-Report Rate** | SELF / (SELF + USER) × 100 | ≥90% |
| **Total #BadAgent** | Count per week | Track trend |
| **Pattern Detection** | Cascade indicators | 4+ = pattern |

## 📋 Overview

L-232 establishes a mandatory security gate for the AnythingLLM DrupalWiki connector. CVE-2026-24478 (Path Traversal → RCE) affects AnythingLLM < v1.10.0. The connector MUST NOT be enabled until version verification passes. Ecosystem standard = v1.11.2 (#LatestStableVersion).

| Risk | Consequence |
|------|-------------|
| Path Traversal → RCE | Full server compromise |
| Affects | AnythingLLM < v1.10.0 |
| Attack vector | DrupalWiki connector endpoint |
| **INT-P02 status** | **❌ v1.9.1 VULNERABLE — @SHD upgrading** |
| **Lite.OCPA.Group status** | **❌ v1.9.0 VULNERABLE — @SHD upgrading** |
| #FedArch impact | ALL 14 instances — connector NOT deployed (🟡 LATENT) |

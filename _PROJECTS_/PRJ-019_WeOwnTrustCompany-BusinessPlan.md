## 2.1 Purpose of This Application

We Own Trust Company, Applicant ("the Company") respectfully submits this business plan as part of its application to the Nevada Financial Institutions Division ("FID") for a charter to operate as a public trust company under Nevada Revised Statutes Chapter 669.

The Company seeks to provide custody services for self-directed individual retirement accounts ("SDIRAs"), enabling account holders to invest in alternative assets through an IRA-owned single-member limited liability company ("LLC") structure.

## 2.3 Core Business Model

The Company operates a straightforward custodial model:

1. **Members** open self-directed IRA accounts with the Company.
2. **Each IRA** holds one single-member LLC, owned by the IRA.
3. **The LLC** invests in IRS-permissible alternative assets including real estate, promissory notes, private equity, precious metals, tax liens, and other qualifying investments.
4. **The Company** provides custody of the LLC, disclosures, education, and regulatory compliance — NOT investment advice and NOT asset monitoring.

```
Member → Opens SDIRA → SDIRA owns LLC → LLC invests in assets

┌──────────┐    ┌──────────────┐    ┌─────────────┐    ┌──────────────┐
│  Member  │───→│  SDIRA       │───→│  Single-    │───→│  Alternative │
│          │    │  (Company =  │    │  Member LLC │    │  Assets      │
│          │    │  Custodian)  │    │  (IRA-owned)│    │  (IRS-       │
│          │    │              │    │             │    │  permissible)│
└──────────┘    └──────────────┘    └─────────────┘    └──────────────┘

Company role: CUSTODIAN ONLY
- ✅ Custody of LLC
- ✅ Disclosures + education
- ✅ Regulatory compliance
- ✅ IRS reporting (Form 5498, 1099-R)
- ❌ NOT investment advice
- ❌ NOT asset monitoring
- ❌ NOT a broker-dealer
```

### IRS-Permissible Assets in SDIRAs

| Asset Category | Permissible? | Examples |
|----------------|-------------|---------|
| Real estate | ✅ YES | Residential, commercial, raw land, rental |
| Promissory notes | ✅ YES | Private lending, mortgage notes |
| Private equity | ✅ YES | LLC interests, LP interests, private stock |

| Tax liens / tax deeds | ✅ YES | Municipal tax lien certificates |

| Life insurance | ❌ NO | Life insurance contracts owned by IRA (IRC §408(a)(3)) |

### Prohibited Transactions (IRC §4975)

| Prohibited Transaction | Description |
|-----------------------|-------------|
| Self-dealing | IRA owner cannot personally benefit from IRA assets |
| Transactions with disqualified persons | Cannot transact with family, fiduciaries, or >50% owned entities |
| Personal use of IRA assets | Cannot live in, use, or benefit from IRA-owned property |
| Lending to/from IRA | IRA owner cannot lend to or borrow from the IRA |
| Providing services | IRA owner cannot provide services to IRA-owned entities (limited exceptions) |

## 6.2 Single-Member LLC Structure

### Why an LLC?

| Benefit | Description |
|---------|-------------|
| **Checkbook control** | Member directs investments through LLC bank account |
| **Asset protection** | LLC provides liability protection for IRA assets |
| **Flexibility** | LLC can hold multiple assets |
| **Privacy** | LLC name on title, not IRA or member name |
| **Simplicity** | One LLC per IRA — clean structure |

### LLC Governance

| Field | Value |
|-------|-------|
| Owner | IRA (100% — sole member) |
| Manager | Member (as manager — NOT as owner) |
| EIN | LLC obtains its own |
| Bank Account | LLC opens its own |
| Title | Assets titled in LLC name |

### Important: Member ≠ Owner

```
CORRECT:
  IRA (custodied at Company) → OWNS → LLC → OWNS → Assets
  Member → MANAGES → LLC (investment decisions only)

PROHIBITED:
  Member → OWNS → LLC (prohibited transaction)
  Member → USES → IRA assets (prohibited transaction)
```

We Own Trust Company operates on a **cooperative cost-based model**. Fees cover operating costs and build appropriate reserves — they do not maximize profit. As the cooperative grows, fees decrease.

| **Cost-based** | Fees cover costs + reasonable reserve |
| **Transparent** | All fees published; no hidden charges |
| **Decreasing over time** | Scale → lower per-account costs → lower fees |
| **Board-governed** | Board sets fees with member advisory input |
| **Competitive** | At or below industry average |
| **Simple** | B-500 model — no transaction fees, no closure fee |
